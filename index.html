A Long Winter

This is the narration script to an unfinished game made during a weekend long game jam about a man staying in an isolated cabin.
Role: Producer, Writer.
Contributions: Production, Story, Dialogue
I took the lead with this project as it was the first jam that much of the rest of the team was working on. The goal was to make a game in which the player played as the person staying at the cabin in the story. Unfortunately this was slightly too amibitious of a project and while we got a game produced, it was still unfinished by the end of the jam. It was my first time acting as the producer for a weekend project like this and it helped me learn a lot about scoping ultra short term projects.
Engine: Unity
